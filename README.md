# 🧮 Desmos Regression Trainer

A free, browser-based SAT math practice tool that teaches you how to use **Desmos** to solve questions faster — not just get the right answer, but actually understand the regression techniques that'll save you precious minutes on test day.

> *"I made this to help people stop losing points on questions they could've solved in 20 seconds with Desmos. Go practice. The SAT is not that deep." — the developer, probably*

---

## 🎯 What is this?

The SAT allows you to use Desmos during the math section. Most people use it as a basic graphing calculator. This trainer teaches you to use it as a **regression machine** — a way to let Desmos solve equations for you by fitting parameters to data.

If you're not using regressions on the SAT, you're leaving points on the table. This tool fixes that.

---

## 📚 What's covered

Practice questions across 12+ categories, from easy to hard:

- **Linear Properties from Two Points** — slope, x/y-intercepts
- **Trigonometric Identity Evaluation** — sin/cos identities with Desmos in degrees mode
- **Simple & Complex Percentage Relationships** — using the `%` regression trick
- **Circle Regression** — fitting circles with `x₁²+y₁²+ax₁+by₁+c~0`
- **Equivalent Expression Constants** — regressing across all x values
- **Exponential Coefficients** — identifying which form shows a given factor
- **Quadratic System of Equations** — using `[a,a]~[b+c,bc]` style regressions
- **Parabola Symmetry** — vertex and intercepts
- **Radius of Non-Standard Circles** — with parameter p
- **Trinomial & Quartic Factoring** — finding integer factors fast
- **Quadratic and Linear Intersection** — tangency conditions
- **Quadratic Regression Limit** — the trickiest one. good luck. enjoy my tips <3

## 🚀 How to use it

1. Go to the live site (link in the repo description)
2. Select the categories you want to practice
3. Click **New Question** or **Start Test**
4. Solve it in Desmos, type your answer, hit Enter
5. Use **Shift+Enter** to check and immediately load the next question
6. Use **Alt+Enter** to start a timed test

Check the **?** button for keyboard shortcuts and Desmos graphing shortcut links.
